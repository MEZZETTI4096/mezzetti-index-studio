# Mezzetti⁴⁰⁹⁶ Studio — The Index

Estúdio de design autoral. Identidade, sistema, experiência — playable.

🔗 **Live**: https://mezzetti-index-studio.netlify.app

## Sobre

Experiência interativa que serve como portfólio e demonstração técnica do estúdio.
Construído em HTML + CSS + JavaScript vanilla, sem frameworks.

### Recursos
- Hero com tipografia variable reativa ao cursor
- Painel flutuante de controles (HUE, CHAOS, RGB)
- Manifesto em grid compacto com 6 princípios
- **Pet Factory** — editor estilo MS Paint 95 pra criar pets personalizados
- Pets voadores com personalidade (Trevoso, Fofinho, Descolado)
- Bento grid 3D pro portfolio
- Lab com 3 experimentos interativos (gradient mesh, type playground, shape sandbox)
- Formulário de contato com envio dual (WhatsApp / Email)
- Cartão postal com glyph da marca girando

### Stack
- HTML semântico
- CSS3 com custom properties, animations, blend modes
- JavaScript vanilla
- Fontes: Fraunces (display), Bricolage Grotesque (UI), JetBrains Mono

### Deploy
Deploy automático via Netlify a cada push na branch `main`.
